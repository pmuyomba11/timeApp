# TimeApp

A simple web application that displays time for a few cities within the United States.


## Technologies Used
1.HTML5

2.CSS3

3.Javacript

4.JQUERY

5.TimeZoneDB API




## Usage/Examples

```
Example Input: America/Chicago, America/Los_Angeles, America/Detroit and so on..
```
